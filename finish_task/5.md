### 第一步：引入专业字体 (Typography)

代码管理器的灵魂是字体。我们需要配置**无衬Hb线字体（UI用）和等宽字体（代码用）**。

1. **修改 `app/layout.tsx**`
引入 `Inter` (UI) 和 `JetBrains Mono` (代码) 字体。
```tsx
// app/layout.tsx
import type { Metadata } from "next";
import { Inter, JetBrains_Mono } from "next/font/google"; // 1. 引入字体
import "./globals.css";

// 2. 配置字体对象
const inter = Inter({
  subsets: ["latin"],
  variable: "--font-sans", //CSS变量名
  display: "swap",
});

const jetbrainsMono = JetBrains_Mono({
  subsets: ["latin"],
  variable: "--font-mono", //CSS变量名
  display: "swap",
});

export const metadata: Metadata = {
  title: "Snippet Manager",
  description: "Personal code snippet manager",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="zh-CN" className={`${inter.variable} ${jetbrainsMono.variable}`}>
      <body className="font-sans antialiased bg-gray-50 dark:bg-[#09090b] text-gray-900 dark:text-gray-100">
        {children}
      </body>
    </html>
  );
}

```


2. **修改 `tailwind.config.ts**`
让 Tailwind 识别这两个 CSS 变量。
```typescript
// tailwind.config.ts
import type { Config } from "tailwindcss";

const config: Config = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}",
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {
      fontFamily: {
        sans: ["var(--font-sans)", "system-ui", "sans-serif"],
        mono: ["var(--font-mono)", "monospace"], // 代码块将自动使用这个
      },
    },
  },
  plugins: [],
};
export default config;

```

### 第二步：全局样式与细节 (Global Styles)

现在的 `app/globals.css` 比较空。我们需要添加一些全局的“质感”样式，比如**自定义滚动条**（代码块必备）和**文字选中色**。

* **修改 `app/globals.css**`：

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

/* 自定义样式层 */
@layer base {
  /* 选中文本的颜色 */
  ::selection {
    @apply bg-blue-500/30 text-blue-900 dark:text-blue-100;
  }
}

@layer utilities {
  /* 隐藏滚动条但保留滚动功能 (可选) */
  .no-scrollbar::-webkit-scrollbar {
    display: none;
  }
  .no-scrollbar {
    -ms-overflow-style: none;
    scrollbar-width: none;
  }
}

/* 这是一个针对 Webkit 浏览器的全局滚动条美化，特别适合代码块 */
::-webkit-scrollbar {
  width: 8px;
  height: 8px;
}

::-webkit-scrollbar-track {
  @apply bg-transparent;
}

::-webkit-scrollbar-thumb {
  @apply bg-gray-300 dark:bg-gray-700 rounded-full hover:bg-gray-400 dark:hover:bg-gray-600 transition-colors;
}

::-webkit-scrollbar-corner {
  @apply bg-transparent;
}

```

### 总结：如何验证CSS效果

完成上述修改后：

1. **字体**: 代码块应该显示为 `JetBrains Mono`（更清晰的编程字体）。
2. **质感**: 搜索框应该有模糊背景（Glassmorphism），卡片悬浮时会轻微上浮并加深阴影。
3. **细节**: 代码块的滚动条应该是灰色的细条，而不是浏览器默认的粗滚动条。
4. **深色模式**: 切换系统主题，颜色应该平滑过渡，背景有深邃的层次感。

这就是使用 Tailwind CSS 4 进行“生产级”样式构建的完整流程，无需编写传统的 CSS 文件，而是通过配置 Design Tokens 和组合 Utility Classes 来实现设计。
